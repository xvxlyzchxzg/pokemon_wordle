# pokemon_wordle

A Pokemon-themed Wordle-style guessing game built using React and PokeAPI 


How it works: 
- A random Pokemon is selected as the answer
- Players have 6 attempst to guess the correct pokemon name
- 
Green = Correct letter in correct position 
Yellow = Correct letter in wtong position 
grey = Letter not in the Pokemon name 


Additional Features: 
- Progressive sprite reveal after incorrect guesses
- Pokemon typing hitn revealed after 3 incorrect guesses
